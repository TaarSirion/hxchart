Type

The type attribute is one of two mandatory attributes to be set for a chart to work.

Possible values for type are

  • scatter
  • line
  • bar

Data

The data attribute is one of two mandatory attributes to be set for a chart to work.

Data.values

data.values is a Map that can contain technically contain any kind of information, but needs certain keys depending on the chart.

Scatter

Mandatory

  • x:Any
  • y:Any

Optional

  • groups:String

Sidenote: It is technically possible to set both x and y as String, but that will result in a stupid chart, so don't do that.

Line

Mandatory

  • x:Any
  • y:Any

Optional

  • groups:String

Sidenote: It is technically possible to set both x and y as String, but that will result in a stupid chart, so don't do that.

Bar

Mandatory

  • x:Any
  • y:Any

Optional

  • groups:String

Sidenote: It is technically possible to set both x and y as String, but that will result in a stupid chart, so don't do that.

Style

style is an optional attribute. The attributes that can be set in style are all named in a way, so they can be reused in many different charts.

Usually you can set the style attributes in three ways:

  • A single value, in all charts this will result in every manifestation having this value.
  • An array the size of unique groups, in case more than one group is defined, this will result in every manifestation of a group to have the same value.

    Depending on the chart, this setting may make no sense and won't result in anything

  • An array the size of values, this will result in every manifestation to have its own value.

    Depending on the chart, this setting may make no sense and won't result in anything

As an example this may look something like this:

Don't use this example trailinfo for creating a plot, it won't work as type and data are missing. This should only demonstrate how the values can be set

                    
                        var trailinfo = {
                            style: {
                                size: 3 //Single Value
                            }
                        };
                        var trailinfo = {
                            data: {
                                values: ["groups" => ["1", "1", "2", "2"]] // Given that we have two different groups
                            },
                            style: {
                                size: [3, 4] //Array size of groups
                            }
                        };
                        var trailinfo = {
                            data: {
                                values: ["x" => [1,2,3]] // Given that we have three values
                            },
                            style: {
                                size: [3,4,1] //Array size of values
                            }
                        };

Style.size

style.size describes an attribute for setting sizes of different kind. Depending on the chart this varies.

Scatter

For scatter plots style.size describes the radius of the drawn points. It can be set in all three ways described in style.

Line

For line plots style.size describes the thickness of the drawn line. It can be set as a single value or as an array of groups.

Style.colorPalette

style.colorPalette describes the colors used in a chart.

Style.alpha

style.alpha describes the alpha value used in a chart.

Style.groups

style.groups describes the groups used in a chart. Don't use this, if you don't know what it does.

Style.borderStyle

style.borderStyle describes the styling for the borders of a chart. Depending on the chart this may do nothing. There are three subattributes that can be set

Style.borderStyle.color

style.borderStyle.color describes the color of the borders of a chart.

Style.borderStyle.alpha

style.borderStyle.alpha describes the alpha value of the borders of a chart.

Style.borderStyle.thickness

style.borderStyle.thickness describes the thickness of the borders of a chart.
